About Video Game Data

The data used in this website are all found in the CORGIS dataset project, and is found here. The data found in this list ranges from 2004 to 2008. The information on the dataset was gathered from crowd-sourced data on "How Long to Beat".

One of the statistics that is listed is whether a game is licensed or not. These licenses include a copyright license, which allows use of copyrighted assets like music, artwork and other things, trademark license, which allows use of a trademarked name, logo or brand, patent license, which allows use of patented technology, and a content license, allows devs to use certain characters or storylines.
